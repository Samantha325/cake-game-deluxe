# 蛋糕製作遊戲 (Cake Making Game) 規格說明書

## 專案概述
這是一個互動式的蛋糕製作遊戲，玩家需要按照食譜順序選擇正確的食材來完成蛋糕。遊戲強調視覺效果和用戶體驗，並包含計分和獎勵機制。

## 已實作功能詳細說明

### 遊戲核心機制
- 食材選擇系統
  * 每次隨機打亂食材位置
  * 固定食譜順序：麵粉->雞蛋->牛奶->糖->奶油->香草精
  * 錯誤選擇扣5分，正確選擇加10分
  * 3連擊以上得分加倍

### 使用者介面
- 三欄式布局已完成
  * 左側：食譜面板（含進度條）
  * 中間：食材選擇格線
  * 右側：蛋糕展示區

### 視覺效果
- 4階段蛋糕製作過程
  * 使用 Flaticon CDN 圖示
  * 轉場動畫流暢
  * 懸浮效果與陰影

### 技術實作
- HTML5 結構完整
- CSS3 動畫與特效
- JavaScript/jQuery 邏輯
- localStorage 存儲機制
